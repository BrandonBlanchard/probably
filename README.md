There are three users spread across three different roles.

### Astrid the Admin
 - Can create/edit/delete artifacts
 - Can see all teams.

### Felix the TeamMember
- Can edit artifacts
- Only see own team.

### Terrance the Observer
- Cannot edit anything
- Only see own team.
